## I. Install Keras

Reference: https://keras.io/

### i. Activate Target Virtual Environment
```
> activate tensorflow
```

### ii-A. Install with Pypi (pip)
```
> pip install keras
```

### ii-B. Install from Github source
```
> git clone https://github.com/keras-team/keras.git
> cd keras
> sudo python setup.py install
```
